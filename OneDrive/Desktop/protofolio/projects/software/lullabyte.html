Lullabyte — AI Infant Vocalization Classifier

Project Summary

Real-time infant vocalization classification system to detect baby needs using audio analysis and a lightweight model served by Flask.

Components: Mobile frontend (React Native / Flutter prototype), Flask API, ML model (Keras/TensorFlow, MFCC preprocessing)

Outcome: Near real-time inference with high precision for key infant sounds.

Read full description

Lullabyte — Details

Audio is preprocessed into MFCC features and passed to a lightweight CNN. The model is served by a Flask endpoint that returns predictions in JSON. The mobile client streams short audio clips to the server and receives prompt classifications.

Tech

  • Python, librosa (audio), TensorFlow/Keras
  • Flask API for model inference
  • Mobile prototype for demo and UX
