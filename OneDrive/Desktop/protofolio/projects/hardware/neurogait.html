Neuro:Gait Feedback Wearable

Project

Wearable using MPU6050 + Arduino + Bluetooth to collect gait data and provide real-time feedback for mental health / rehabilitation support.

Stack: Arduino Nano, MPU6050 IMU, Bluetooth module, mobile interface for visualization & alerts.
